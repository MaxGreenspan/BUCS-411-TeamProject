A user wants to find a previously generated quote or image.
The user logs into the website, and clicks the history button on the top right.
Then, it shows a list of the user's history.
The user sees the history id and the keyword that is used as well as the date it is generated, and then realizes one history is of interest.
The user clicks the "view" button on the right to check the details.
The quote and image are shown just like a result of a new search.
